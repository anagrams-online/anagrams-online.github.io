# Anagrams Online

## Play in the Browser
Click [here](https://anagrams-online.github.io) to open the browser game.

## Download Locally
Link **blocked** or want to **edit** the game files?
1. Download Git for Windows [here](https://git-scm.com/install/windows).
2. Finish Git installation.
3. Make a new folder on your desktop.
4. Open the new folder in a command prompt.
5. Run this code: `git clone https://github.com/anagrams-online/anagrams-online.github.io.git`
6. Git will make a new folder inside the folder you made. Open this new folder.
7. Find the file named **index.html** and double-click it to open the browser game.
